Badge component

Badges are used to highlight an item's status for quick recognition.

Status Badge

To use status badge add class name badgeto a div tag which is inside a div with a class name badge-containeralong with an avatar component. For different status add class name accordingly success, offline, error.

Icon Badge

To use icon badge add class name badge-iconto a div tag which is inside a div with a class name badge-container along with an icon. For different status add class names accordingly success, offline, error.
