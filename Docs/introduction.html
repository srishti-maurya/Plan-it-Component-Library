Introduction to Plan it

Plan it is a CSS framework packed with classes that enables developers to leverage the best styles and components by using pre-defined classnames. Plan it assists you in constructing your project by putting a strong emphasis on functionality, as we take care of your designs.

Installation guide

Copy the link and paste it in the headtag of your html, above all other style sheets. i.e. import the stylesheet in your html file.
